# Cobblemon Casino

Bring a real casino atmosphere to your Cobblemon world.

Cobblemon Casino is a NeoForge `1.21.1` addon for **Cobblemon** that adds slot machines, blackjack tables, gachapon machines, casino workers, themed decorations, casino currency, and in-world records to build a full casino experience for your server or modpack.

## Credits

Cobblemon Casino is based on the original **Casino Rocket** mod by **AndresPR512** which made a great work ! I ported it to neoforge and made several fixes, changes and additions to it, so do not consider it a 1:1 version.

## Features

- Playable **slot machines** with configurable bets, lines, and payouts
- **Blackjack tables** with dedicated casino gameplay and shared player balance support
- **Chip tables** for casino currency exchange
- **Item gachapon** and **Pokemon gachapon** machines
- Plushies-style gachapon support using Pokeblocks plushie rewards by default
- **Casino worker villagers** and configurable casino trade content
- **Casino scoreboard** for player records and leaderboard-style stats
- Decorative casino chip blocks and themed casino building content
- Chip-based casino economy with wallet support

## Built for Cobblemon servers and packs

Cobblemon Casino is designed to feel at home in a Cobblemon pack, server or in your normal survival world.

It uses Cobblemon content directly for rewards, themed machines, villager integrations, and default gameplay loops, while still giving server owners room to customize the economy, rewards, rates, and machine behavior. Change it the way you like it !

## Commands

Main command root:

- `/cobblemoncasino`

Included command groups cover:

- casino villager spawning and setup
- gachapon rates, stats, and leaderboards
- slot machine rates and leaderboards
- blackjack leaderboards

## Configuration

The mod generates its config files automatically on first launch.

Config folder:

```text
config/cobblemoncasino/
```

If you want to regenerate fresh defaults for any reason, remove the generated Cobblemon Casino config files and start the game or server again.

## Dependencies

Required:

- NeoForge `21.1.200+`
- Minecraft `1.21.1`
- Cobblemon
- Cloth Config API

Optional:

- CobbleDollars
- Pokeblocks

CobbleDollars is not required, but it can be used for optional money-based integrations depending on your setup.
Pokeblocks is not required either, but it is if you want the plushies gachapon machine to use its default plushie reward pool.
